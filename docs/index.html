Welcome to Pondasee Docs

Welcome to the Pondasee documentation, here you can learn what Pondasee includes and how to use it as your starting point.

What is Pondasee?

Pondasee is front-end starter kit combined with the power of SCSS & Compass, we made Pondasee to help front-end or developer to speed up their work-flow in making web application.

Quick Start

  • Install Compass, you can read on Compass official site, here
  • Or update your Compass to the latest version gem install compass
  • Clone this project git clone git://github.com/tokokoo/pondasee.git
  • Or you can download the zip file here
  • Compile pondasee project folder and you're ready to go!
